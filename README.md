## Campaign Details

- **Offer:** 8-week Summer Special — $118
- **Includes:** Full gym access + 2 × 30-min PT sessions + SGT classes + 39 GX classes
- **Retail value:** $375
- **Booking:** WellnessLiving widget embedded in signup section

---

## Tech

- Single-file HTML/CSS/JS — no build tools, no frameworks
- Drops straight into WordPress as a standalone page
- Mobile-first responsive (360px → 600px → desktop)
- Fonts: Fraunces · Caveat · Plus Jakarta Sans (Google Fonts)
- Brand: `rgb(42, 49, 146)` indigo + `rgb(165, 245, 35)` lime
